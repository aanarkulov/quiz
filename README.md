# Quiz App

# Описание проекта

Приложение для прохождения тестов. Есть возможность регистрации в системе.
Авторизованный пользователь может добавлять тесты. Добавленный тест будет 
отображен на главной странице в конце списка тестов. В один тест можно добавлять 
сколько угодно вопросов с четырьмя вариантами ответов для каждого вопроса.

После выбора теста открывается первый вопрос с вариантами ответов, в случае
выбора неправильного ответа переход не следуйщий вопрос ну будет осуществлен, до тех пор
пока не будет выбран правильный ответ, для результата будет записан первый выбранный ответ.
После последнего вопроса будет отабражен рузультат теста.

Посмотрите это приложение в действии:

[https://quiz-2019a.firebaseapp.com/](https://quiz-2019a.firebaseapp.com/).

# Запуск (Local Server)

### Старт

```sh
$ git clone https://github.com/aanarkulov/quiz.git
$ cd quiz
$ npm install
$ npm start
```

> Вы можете использовать [yarn](https://yarnpkg.com/) вместо [npm](https://npmjs.com).

### Для запуска тестов

В корне проекта

```sh
$ npm test
```

Показ отчета покрытия тестов

```sh
$ npm test --coverage
```
